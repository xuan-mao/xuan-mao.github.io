climb the great wall

v2raya + v2ray

  • enable repository archlinuxcn
# /etc/pacman.conf
# add the two lines below
[archlinuxcn]
Server = https://mirrors.ustc.edu.cn/archlinuxcn/$arch

# exec the following line in zsh
pacman -S archlinuxcn-keyring
  • install v2raya + v2ray
pacman -S v2ray v2raya
  • enable v2ray.service and v2raya.service
systemctl enable --now v2ray.service
systemctl enable --now v2raya.service
  • config v2raya by webUI at http://localhost:2017/
